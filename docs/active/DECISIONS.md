# DECISIONS.md — Noteflex 결정 로그

> 추가만 하는 결정 로그. 한 번 기록된 결정은 수정·삭제하지 않음.
> 형식: 날짜 · 결정 · 이유 · (있으면) 버린 대안

---

## 2026-06-05

### 언어 자동 감지에서 타임존 신호 제거
**이유**: 위치는 언어 선호가 아님 — 한국 거주 영어 기기 사용자 오판 제거. 기기 언어(`navigator.language`)만 사용.

---

### 페이지 타이틀·OG 메타 영어 고정
**이유**: 링크 미리보기 크롤러는 유저 언어를 따라갈 수 없음. 정적 HTML만 읽는 크롤러에게 언어별 동적 메타는 의미 없음.

---

## 2026-06-06

### flowType `'implicit'` 명시 고정
**이유**: 매직링크는 교차 기기 클릭(폰 요청→데스크톱 메일)이 흔해 PKCE(요청 브라우저의 verifier에 묶임)와 상극. `implicit`이 유일하게 안전한 선택.

---

### 매직링크 + 6자리 코드 하이브리드 채택
**이유**: 링크는 행복 경로(한 번 탭). 코드는 스캐너 선소비·교차 기기 면역 비상구. 5/31 봉인 재발 방지.
**버린 대안**:
- OTP 단독 — 매 로그인마다 6자리 입력 마찰
- 링크 단독 — 5/31 실패 재발 위험 잔존

---

### 인증 실패 무음 처리 금지 원칙
**이유**: 모든 실패는 유저에게 보이고 다음 행동(재발송·코드 입력 등)으로 연결해야 함. 5/31 봉인 사건 — 링크 오류가 무음으로 `/`로 리다이렉트되어 가입 불가 상태가 수일간 발견되지 않은 교훈.

---

### AuthCallback opener 분기
**이유**: Google OAuth는 전체 리다이렉트(opener 없음) → 즉시 navigate 필요. 팝업만 `window.close()` 시도. 기존 코드는 전체 리다이렉트에서도 "탭을 닫으세요" 안내를 내보내는 오류 있었음.
**버린 대안**: 항상 navigate — 팝업 탭이 닫히지 않고 홈으로 이동하는 UX 불일치

---

### 신규 가입 판정 = `user.created_at` 10분 이내
**이유**: 환영 페이지(`/welcome`)는 신규에만 노출되어야 하며, 재방문·기존 로그인과 구분해야 함. 10분 임계값은 콜백 처리 지연을 충분히 커버.
**구현 위치**: AuthCallback, AuthBroadcastListener, AuthModal verifyOtp, WelcomePage guard — 4곳 동기화.

---

### Apple 로그인 보류
**이유**: PWA는 Apple 로그인 의무 아님. 네이티브 앱 출시 시점에 $99 개발자 계정과 함께 도입하는 것이 효율적.
**버린 대안**: 지금 도입 — PWA만 운영 시 이점 없음, 구현 복잡도만 추가

---

## 2026-06-07

### 결제 MoR Plan B = Creem 확정
**이유**: 한국 판매자 로컬 이체 지급 지원(공식 문서 확인) + $4.99 저가 구독에서 수수료 최저(3.9% + $0.40, Paddle 대비 건당 ~3%p 절감) + 셀프서브 온보딩.
**주의**: 2024년 출범 신생(거래상대 리스크). 지급 최소 금액 $7 → 잔액 누적 후 지급 운용.
**버린 대안**:
- Lemon Squeezy — Stripe Managed Payments로 흡수 중 + 한국 판매자 경로 불확실
- FastSpring — 견적제 + 고정 수수료 $0.95가 $4.99 결제의 ~25% 잠식
- Polar — 2026 개편 후 수수료 이점 없음

---

### KYC 서류 원칙
**결정**: 자동 KYC에는 표준 국문 원본을 제출한다.
**이유**: 영문 번역 등본(비표준 양식)은 자동 분류기에서 인식 실패 — Sumsub·Paddle 공통 증상으로 확인. 표준 조합: 여권(신원) + 국문 등본(주소 증빙).

---

### 결제 1순위 = Creem 확정 + HN 주간 결제 코드 동결
**결정**: Creem 계정 승인 완료(2026-06-08). 결제 1순위 = Creem (수수료·한국 이체 우위). Paddle 휴면 백업. 결제 통합은 HN 이후 진행 — 최대 트래픽 시점에 결제 코드 급출시 금지(이중청구·권한실패 리스크 > 즉시 전환 이득).

---

### Mastery 카드 정보 공개 정책 수정 (B-0 보조 항목)
**결정**: 기준 4개 이름·달성여부(✓/–)·'Clear at 100' 라벨은 모든 티어 공개. 수치(예: 91%/85%, 3/10)만 비Pro 블러 유지.
**이유**: 승리 조건의 명료함은 게임 사용성(무료), 진단의 깊이는 프리미엄 가치. 길 잃은 유저는 전환되지 않음.
